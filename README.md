# Python_Diwali_Sales_Analysis
# Diwali Sales Analysis (Python Project)

This project is based on Diwali sales data analysis using Python.  
The main goal is to analyze customer buying behavior and identify sales trends during the Diwali festival season.

---

## 📌 Project Objective
- Analyze Diwali sales dataset
- Understand customer demographics
- Identify top performing states, occupations, and product categories
- Find purchasing patterns based on age and gender
- Generate useful business insights for marketing and sales improvement

---

## 📊 Dataset Information
The dataset contains Diwali sales details such as:
- User_ID
- Cust_name
- Product_ID
- Gender
- Age Group
- Age
- Marital_Status
- State
- Zone
- Occupation
- Product_Category
- Orders
- Amount

---

## ⚙️ Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook / Google Colab

---

## 🔍 Steps Performed
1. Imported required Python libraries
2. Loaded Diwali sales dataset
3. Cleaned data (removed null values, corrected datatypes)
4. Performed Exploratory Data Analysis (EDA)
5. Visualized sales patterns using charts
6. Generated business insights

---

## 📈 Analysis Performed
- Gender wise sales analysis
- Age group purchasing behavior
- State wise total sales
- Occupation wise sales
- Product category analysis
- Top customers and high sales regions

---

## ✅ Key Insights
- Most active buyers during Diwali
- Best performing states in terms of sales
- High selling product categories
- Occupation groups with highest spending
- Age groups contributing maximum revenue

---

## 🚀 How to Run This Project
1. Clone this repository or download the files

4. Run all cells to see output and graphs
